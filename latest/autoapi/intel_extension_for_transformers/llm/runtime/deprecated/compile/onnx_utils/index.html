intel_extension_for_transformers.llm.runtime.deprecated.compile.onnx_utils.bias_to_int32(bias_node, a_scale, b_scale)[source]

Convert the int8 bias to int32 bias.

Parameters:

  • bias_node – bias_add in graph (from onnx framework)

  • a_scale – matmul node input matrice a scale tensor

  • b_scale – matmul node input matrice b scale tensor

  • model – Grpah class

Returns:

int32 bias numpy array

fp32_bias = (int8_bias - int8_bias_zero_point) * int8_bias_scale int32_bias = fp32_bias / (a_scale * b_scale)
