intel_extension_for_transformers.neural_chat.tools.rome.repr_tools

Contains utilities for extracting token representations and indices from string templates.

Used in computing the left and right vectors for ROME.

Given list of template strings, each with one format specifier (e.g. “{} plays basketball”), and words to be substituted into the template, computes the post-tokenization index of their last tokens.

We use left-padding so the words idxs are negative numbers.
